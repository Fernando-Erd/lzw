## LZW Algorithm

This program implements the lzw algorithm.

##### To Compile:
```bash
Compactor - make PandaCompact
Decompressor - make PandaDescompact
```

##### Clear files

```bash
make clean
make clean_txt
```

##### To execute:

```bash
./PandaCompact <input.txt> <output.txt>
./PandaDescompact <input.txt> <output.txt>
```

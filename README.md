## LZW Algorithm

This program implements the lzw algorithm.

##### To Compile:
Compactor
```bash
make PandaCompact
```
Decompressor
```bash
make PandaDescompact
```

##### Clear files

```bash
make clean
make clean_txt
```

##### To execute:

```bash
./PandaCompact <input.txt> <output.txt>
./PandaDescompact <input.txt> <output.txt>
```
